## Giới thiệu sơ lược
Báo cáo và source code cho bài tiểu luận giữa kỳ môn học Lập trình web và Ứng dụng.

Được xây dựng bằng:
- C# (ASP.NET Core)
- Microsoft SQL Server
- Bootstrap
- jQuery
- Azure Server
